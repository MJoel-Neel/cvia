# CVIA — Page pédagogique sur l'intelligence artificielle

Une page web pédagogique qui explique simplement l’intelligence artificielle : ses bases, ses applications et ses enjeux.  
Le projet est réalisé en HTML, CSS et JavaScript et est déployé via GitHub Pages.
